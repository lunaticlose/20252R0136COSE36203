# 20252R0136COSE36203
한국어 가짜 뉴스 판별 프로젝트

## 1. 프로젝트 개요
한국어 뉴스 기사의 제목과 본문을 분석하여 가짜 뉴스와 진짜 뉴스를 판별하는 딥러닝 모델을 개발하는 것을 목표로 합니다.

## 2. 사용 모델 및 방법론
1. **Baseline**: TF-IDF + Logistic Regression
2. **Proposed**: KoBERT Fine-tuning

## 3. 파일 설명
- `news_predict_progress.ipynb` : 중간과정에 사용한 코랩 코드
- `news_predict_fianl.ipynb` : 최종적으로 사용한 코랩 코드
- `requirements.txt`: 실행에 필요한 라이브러리 목록
- `Final_Report.pdf`: 최종 프로젝트 보고서
- `train_data.csv`: dacon.io에서 다운 받은 파일로 따로 업로드 하지 않았음
- 출처 : https://dacon.io/competitions/official/235747/data
